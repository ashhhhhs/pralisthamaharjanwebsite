# Security Policy

## Reporting a Vulnerability

If you find a security issue in this portfolio site, please email:

pralisthamaharjan21@gmail.com

Include a clear description, reproduction steps, affected files or URLs, and any screenshots or logs that help explain the issue.

## Scope

This is a static portfolio website. Security-sensitive changes should avoid inline scripts, untrusted third-party embeds, unknown external assets, and forms that collect personal data without a trusted backend.
